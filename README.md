# Graph-Based Route Optimization

## Overview  
This project models a network of cities using graph data structures and computes all possible paths between them to determine the most cost-efficient route. The goal is to simulate real-world route optimization problems and apply algorithmic thinking to minimize travel cost.

---

## Features  
- Represents cities and connections using graph structures  
- Generates all possible paths between a set of cities  
- Calculates total cost (distance) for each path  
- Identifies and outputs the most cost-efficient route  
- Displays all evaluated paths along with the optimal solution  

---

## Technologies Used  
- C++  
- Graph data structures  
- Recursion / path traversal algorithms  

---

## Key Concepts Demonstrated  
- Graph traversal and pathfinding  
- Recursive exploration of all possible routes  
- Cost optimization and comparison  
- Algorithmic problem solving
